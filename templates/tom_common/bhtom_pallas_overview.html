BHTOM-PALLAS provides tools for planning and supporting observations of comets and asteroids.

Ephemeris

This tab queries NASA JPL Horizons via astroquery, allows an MPC observatory code, lets the user define a custom time span and step size, select returned quantities, and generate ephemerides for comet and asteroid targets.

Observation planning

This tab calculates sky motion and maximum exposure time from JPL Horizons sky-plane motion components and an allowed trailing value.
