Photometry

Standard BHTOM photometry assumes that the target position is fixed on the sky, so the same location can be measured consistently from image to image.

For moving targets such as comets and asteroids, that assumption breaks down because the target position changes between frames as the object moves across the detector.

The intended PALLAS approach is therefore to use the image timestamp together with an ephemeris to determine the target position separately for each frame before measuring the photometry.

ATLAS forced photometry

Submit an MPC-recognised moving-object identifier to the public ATLAS forced photometry service. This first version uses the ATLAS moving-object workflow by MPC name and returns the light-curve table directly into BHTOM-PALLAS.

ATLAS expects MPC-style identifiers for moving objects. For asteroids this often means a numbered designation such as (12267) or a packed designation. For comets, use the MPC packed designation.
